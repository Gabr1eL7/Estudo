Exemplos de listas

Lista numerada

Para utilizar lista do tipo numerada é utilizado (ol) junto com (li)

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5
  6. Item 6

NOTA

Podemos utilizar variações de classifição utilizando o type:
(1)= 1,2,3,4,5... (A)= A,B,C,D... (I)= I,II,III,IV,V...

Por exemplo

  1. Item 1
  2. Item 2
  3. Item 3

Lista não numerada

Caso não seja necessário a numeração e ordem é utilizado(ul) junto com (li)

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

Lista Mista

Podemos tambem mesclar uma lista numerada com uma não mumerada, funcionado com uma sub lista filha

  • Item 1
  • Item 2
    1. Item 1
    2. Item 2
    3. Item 3
  • Item 3
  • Item 4
  • Item 5
